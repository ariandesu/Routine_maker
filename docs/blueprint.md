# **App Name**: Timetable Weaver

## Core Features:

- Interactive Schedule Editor: Enable users to build their schedule through an interactive grid interface.
- Template Selection and Customization: Offer multiple visually distinct templates to help bootstrap design. Let the user control fonts and colors of each template.
- Export Functionality: Generate a static image in PNG, JPG, and PDF.
- Import Functionality: Allow importing schedules via a CSV file upload.
- Schedule Sharing: Schedule sharing is achieved via generating a unique URL
- Real-time Preview: Provide a real-time preview of the final schedule image.

## Style Guidelines:

- Primary color: Indigo (#4B0082) to evoke feelings of intellect.
- Background color: Light gray (#E6E6E9), offering a subtle backdrop.
- Accent color: Yellow-Gold (#D4AF37) for highlighting important sections and CTAs.
- Body and headline font: 'Alegreya', a humanist serif with an elegant, intellectual, contemporary feel.
- Responsive grid layout adaptable to desktop, tablet, and mobile devices, ensuring a consistent user experience across platforms.
- Use clear, concise labels and descriptions.
- Simple, professional icons from a standard library.
